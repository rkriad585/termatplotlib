# Installation

## From PyPI

```bash
pip install termatplotlib
```

## From source

```bash
git clone https://github.com/rkriad585/termatplotlib
cd termatplotlib
pip install .
```

## Requirements

- Python >= 3.6
- No external dependencies — uses only the Python standard library

## Verify installation

```python
import termatplotlib as tpl
print(tpl.__all__)
```
